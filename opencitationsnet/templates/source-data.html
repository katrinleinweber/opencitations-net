Source data

The source data for this project is available from our download area.

Data formats

Data processing happens using BibJSON datasets. The datasets are packed into gzipped tarballs and processed by as a stream by a Python script. Each tarball potentially contains millions of files, making it unwise to extract onto disk.

The original source data can be downloaded from the National Library of Medicine's FTP site. The files with names of the format articles.X-X.tar.gz contain NLM XML for all articles in the open-access subset, and were used as the basis for the data here. For more information, see their README.

We provide our data at the following stages of processing:

Raw BibJSON (articles-raw.bibjson.tar.gz)
The article data after a simple conversion from NLM XML.
Sanitized BibJSON (articles-sanitized.tar.gz)
At this stage the data has been augmented with bibliographic data from the Entrez API and some errors corrected.
Unified BibJSON (articles-unified.bibjson.tar.gz)
Records which we believe represent the same entity have been grouped into BibJSON datasets.
Final N-Quads RDF (articles.nq.gz)
Final RDF serialized as N-Quads and gzipped.

Sample data

This website contains roughly a tenth of the data available from the ope-access subset of PubMed Central. Specifically, it contains data contained in open-access articles from the following journals:

  • BMC Bioinformatics
  • BMC Infectious Diseases
  • BMC International Health and Human Rights
  • BMC Medical Research Methodologies
  • Journal of Biomedical Semantics
  • Nucleic Acids Research
  • PLoS Computational Biology
  • PLoS Medicine
  • PLoS Neglected Tropical Diseases
